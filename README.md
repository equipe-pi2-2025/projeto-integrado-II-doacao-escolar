# Projeto-integrado-II-doacao-escolar
Plataforma web de doação e reaproveitamento de materiais escolares

## Sobre o projeto

Este projeto consiste em uma plataforma web para doação e reaproveitamento de materiais escolares, desenvolvida como MVP da disciplina de Projeto Integrado II e Programação Orientada a Objetos.

## Classes implementadas
- **Usuario**: que são os usuários da plataforma, podendo ser do tipo doadores ou beneficiários.

## Possíveis usos da nossa solução
Esta plataforma pode ser utilizada por escolas públicas, organizações sociais e instituições que precisam conectar pessoas que precisam e pessoas que estão dispostas a realizar a doação.
